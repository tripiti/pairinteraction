Detailed Description

SQLite Statement handle.

This object wraps an SQLite statement. It is iterable in a while loop with the step() function or in a range-based for loop.

iterator sqlite::statement::end ( )
inline

returns an empty iterator

The iterator for the SQLite statement is a forward iterator which is implemented in terms of the step() function. Since the step() function will determine the end by itself, this iterator is merely a sentinel.

Returns
empty iterator
